# Careflix
Proyecto para visualizar videos médicos on-demand, categorizados por especialidad.

Tecnologías utilizadas:
* React (front-end)
* MongoDB
* Nodejs (back-end)

El proyecto permitirá:
* Hacer un registro de usuario
* Evitar registros duplicados (mediante la cédula profesional)
* Navegar un home con thumbnails de videos de medicina
* Los videos estarán categorizados por especialidades. 
* La especialidad del doctor loggeado debe coincidir con la categoría seleccionada al entrar
* Abajo de cada video se muestran otros videos similares, sugeridos al estilo de "comprados frecuentemente" de Amazon

Extra:
* Muestra un historial de reproducciones del usuario
* Muestra una sección de "contenido para ti", con sugerencias tanto de la especialidad del doctor como de otras
especialidades.
